# 台股籌碼雷達

台股個股籌碼分布分析工具，以成交量分布（Volume Profile）視覺化主力成本、套牢籌碼與獲利籌碼。

**線上版本：** https://kllin1221.github.io/tw-stock-chip/

---

## 功能

- **成交量分布圖（Volume Profile）** — 右側條狀圖顯示各價位累積成交量
- **POC 主力成本** — 成交量最大的價位
- **價值區（VA 70%）** — 涵蓋 70% 成交量的價格區間（VAH / VAL）
- **獲利籌碼 / 套牢籌碼** — 現價以下 / 以上的持倉比例
- **籌碼集中度** — 前三大量價區間佔總成交量比
- **區間選取** — 拖曳走勢圖選取特定時間段，動態重算籌碼分布

### 動態分級邏輯
- 股價 ≥ $100：每格 $5，最少 50 格
- 股價 < $100：每格 $1，最少 50 格

## 本地執行

直接開啟 `index.html` 即可使用（FinMind 模式無需後端）。

### 啟動 TWSE 代理伺服器（選用）

```bash
node server.js
```

伺服器啟動於 `http://localhost:3000`，接著在頁面設定切換至「TWSE 代理」模式。

```
GET http://localhost:3000/api/stock?symbol=2330&days=240
```
